== Description ==

Call me spoot is a WordPress plugin which puts a Feedback button on your website. Once the user clicks the feedback button, a modal popup appears on the screen. After completing the fields you riceve e-mail notification, telegram messenger notification.

Features:
 
 - Works with any theme!

 - Customizable button!

 - E-mail notification.

 - E-mail address verification.

 - Easy user interface.

 - Create forms & buttons without knowing HTML or PHP.



== Installation ==



1. Copy the plugin folder to / wp-content / plugins /

2. Activate the plugin

3. Use the Settings -> Call me spoot
